📌 PODSUMOWANIE PROJEKTU – AI TEXT DETECTOR (INFO DLA ZESPOŁU)
1. Jak uruchomić aplikację (krok po kroku)
🔹 Backend (API + ML)

Przejdź do katalogu głównego projektu:

ai_text_detector


Aktywuj środowisko:

venv\Scripts\activate


Uruchom backend:

python -m uvicorn app.main:app --reload


Sprawdź:

API: http://127.0.0.1:8000

Dokumentacja: http://127.0.0.1:8000/docs

Backend MUSI działać przed frontendem.

🔹 Frontend (prosty klient webowy)

Otwórz nowy terminal.

Przejdź do folderu frontend:

cd frontend


Uruchom lokalny serwer:

python -m http.server 5500


Otwórz w przeglądarce:

http://127.0.0.1:5500/


Frontend wysyła zapytania do backendu (/analyze).
